#  Hayak Coffee Menu App

A simple **Next.js** project that displays a coffee shop menu with categories and products.  
This project is for learning purposes and to help you understand how to structure a basic Next.js application.

---

## 📁 Project Structure


app/
└── menu/
└── page.tsx # Main file that contains all the menu page code


>  All the code is currently inside **`app/menu/page.tsx`**.  
> Later, you will refactor it into smaller, reusable components.

---
